# Juego: Bicicleta con obstáculos

Juego sencillo hecho con Pygame. El jugador controla una bicicleta (rectángulo) con las teclas de dirección. Los obstáculos aparecen aleatoriamente desde arriba. Al chocar, el jugador pierde una vida (tiene 3) y se vuelve invulnerable 1 segundo.

## Controles

- Flechas: mover arriba/abajo/izquierda/derecha
- R: reiniciar después de Game Over

## Requisitos

Instala las dependencias e ejecuta:

```bash
pip install -r requirements.txt
python main.py
```
